# react-native-apphive-jitsi

An interface to connect to jitsi sdk

## Installation

```sh
npm install react-native-apphive-jitsi
```

## Usage

```js
import { ApphiveJitsiView } from "react-native-apphive-jitsi";

// ...

<ApphiveJitsiView color="tomato" />
```
